A basic Catan simulator to practice placing your initial placements. I love playing Catan and I want to improve at it, so I built this simulator to practice placing my initial settlements.

Super simple design using pygame.

You choose 3 or 4 players from the menu screen, then a random board gets automatically generated. 
And a random playing order employing the snake draft also gets generated.

You can simply start placing your settlements. The game then evaluates and rates if you had good placements or not, and what could've been more optimized/better placements.

Install pygame by using 'pip install pygame' if you already don't have it.
You can just run this by using 'python main.py' after opening a terminal in the code directory.
